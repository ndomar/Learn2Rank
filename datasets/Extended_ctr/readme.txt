This dataset is an extension to the CTR data from Hao Wang. 
The new information are crawled from citeulike, they are:
1- papers authors: authors.csv
2- papers keywords: keywords.csv
3- paper meta info like: citeulike_id, type, journal, booktitle, series, publisher, pages, volume, number, year, month, postedat, address, title, abstract
